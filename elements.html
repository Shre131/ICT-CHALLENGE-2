An HTML file is made of elements. These elements are responsible for creating web pages and define content in that webpage. An element in HTML usually consist of a start tag , close tag and content inserted between them.

Main root:

The HTML element represents the root (top-level element) of an HTML document, so it is also referred to as the root element. All other elements must be descendants of this element. The html element is the root element and it defines the whole HTML document. It has a start tag < html > and an end tag < /html >. Then, inside the < html > element there is a < body > element: The < body > element defines the document's body. Then, inside the < body > element there are two other elements: < h1 > and < i > The < h1 > element defines a heading. It has a start tag < h1 > and an end tag < /h1 > The < i > element defines a part of text or content inside is typically displayed in italic. It has a start tag < i > and an end tag < /i >
